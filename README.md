# Quiz-App


Technologies used
----------------

- Java
- spring-boot
- maven
- This service uses: H2 Memory Database

How to run this service
----------------------

``````
$ mvn clean install
$ mvn spring-boot:run
``````

- URL for the Database Memory console: http://localhost:8083/h2-ui
- username: sa
- password: password
- JDBC URL: jdbc:h2:mem:quiz-db
